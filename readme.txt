== Description ==

ClassicPress-Plug-in, das einen neuen Pen erstellt, der einen Styleguide für dein [UpFront](https://upfront.n3rds.work/upfront-themes/) Design enthält. Extrahiert Themenfarben und Typografieeinstellungen.
